# namel3ss Website

This repository contains the official website for namel3ss, an English-first, AI-native programming language. The language emphasizes deterministic execution, explicit AI boundaries, and built-in explainability.

## What this repository contains

- Website source code (React + Vite)
- Static assets (images, fonts)
- Site content and components
- Build configuration

## Local development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173` (or the next available port).

## Build

Generate a production build:

```bash
npm run build
```

Output will be in the `dist` directory.

## Preview / deploy

Preview the production build locally:

```bash
npm run build
npx vite preview
```

The site can be deployed to any static hosting service:
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
